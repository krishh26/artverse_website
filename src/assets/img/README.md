# Images Folder

Place your images here, including:

- `logo.png` - Your main logo file for the navbar
- Other images for workshops, festivals, artists, gallery, etc.

## Logo Requirements

- **File name**: `logo.png`
- **Recommended format**: PNG with transparent background (or SVG)
- **Recommended dimensions**: 200px width, auto height
- **Content**: Art Verse logo with yellow sun, light blue accent bar, and "THE FOLK FEST" text

## Usage

Images in this folder can be referenced in your components like this:
```html
<img src="assets/img/your-image.png" alt="Description">
```
